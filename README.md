# API REST - H.C.C (Handball Club de Comines)

API REST pour la gestion du club de handball de Comines : adherents, matchs et actualites.

## Technologies

- **NestJS** - Framework backend
- **SQLite** - Base de donnees locale
- **TypeORM** - ORM
- **JWT / bcrypt** - Authentification et hashage

## Installation

```bash
npm install
npm run start:dev
```

L'API demarre sur `http://localhost:3000`.
